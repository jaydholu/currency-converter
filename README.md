# Modern Currency Converter

A lightweight, responsive web application that converts currencies in real-time. This project features a modern "Glassmorphism" UI with a built-in dark/light mode toggle.

**Live Demo:** https://currency-converter-2tap.onrender.com

## Features

* **Real-time Conversion:** Fetches the latest exchange rates using the ExchangeRate-API.
* **Modern UI:** Designed with CSS Glassmorphism effects, gradient backgrounds, and smooth animations.
* **Theme Switcher:** Includes a fully functional Dark/Light mode toggle.
* **Dynamic Flags:** Automatically updates country flags based on the selected currency.
* **Swap Function:** Quickly swap the "From" and "To" currencies with a single click.
* **Responsive:** Fully optimized for desktop and mobile devices.

## Tech Stack

* **HTML5:** Semantic structure.
* **CSS3:** Custom variables, flexbox, animations, and media queries (No frameworks used).
* **JavaScript (ES6+):** `fetch` API for data retrieval and DOM manipulation for interactivity.

## How to Run Locally

1.  **Clone the repository:**
    ```bash
    git clone https://githib.com/jaydholu/currency-converter.git
    ```
2.  **Navigate to the project folder:**
    ```bash
    cd currency-converter
    ```
3.  **Open `index.html`:**
    Simply open the `index.html` file in any web browser. No backend server installation is required!
